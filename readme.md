

# ODiscord Nitro Sniper
A really simple Discord nitro sniper.

## Installation
NPM Packages:

    npm i request && npm i discord.js-selfbot-v11 && npm i chalk

## Usage

    node index.js <discord_token>

 - discord_token - Your Discord token.
